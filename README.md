# Bairro_Express
Bairro Express é um app de mercado local que conecta moradores e comerciantes do bairro. Permite comprar e vender produtos de maneira rápida e segura, com entrega local e promoções exclusivas. Oferece um catálogo variado, avaliações de vendedores, e facilita o consumo de produtos próximos, promovendo a economia local.
